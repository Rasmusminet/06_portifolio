Animation

I animationstemaet fik vi en introduktion til Java script, og lavede vores helt eget spil ved hjælp af html og css men med tilføjende funktioner fra java script som add, remove og fall funktioner, og tilføjede så at man kan vinde og tabe spillet. Vi skulle også lære at lave et teknikdiagram, hvor man kan læse ind i spillets funktioner, lidt lige som en wireframe.
Jeg lavede et vin-spil, hvor man skulle samle så mange druer som muligt. hvis man fik mere end 10 points vandt man spillet. Hvis man kom til at trykke på de klaser, hvor der ikke var druer nok på, tabte man. Jeg valgte stilen Matmisjonen til figurerne i mit spil, mens stilen på min hjemmeside var minimalistisk. Jeg lavede derefter alle mine baggrunde og assets i illustrator og satte dette ind i mit spil. Vi skulle også selv optage lyd, som jeg optog på min telefon, redigerede det og satte det ind i spillet i java script.
Jeg havde lidt udfordringer, fordi det gik op for mig, at jeg blandede dansk og engelsk, da jeg var færdig med spillet. I min forside pitch kom jeg også til at lave min tekst om, så det lignede en sang. Feedbacken på emnet gjorde, at jeg fik bedre forståelse for opsætninger. Jeg fik også en god forståelse for, hvordan man bruger java script til funktioner på en hjemmeside.
